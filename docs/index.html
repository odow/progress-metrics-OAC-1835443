Health of the JuMP ecosystem

This page collates some summary statistics for the JuMP-dev ecosystem.

Count of open issues

Smaller is better. Open issues are a mixture of bugs and feature requests.

Count of open pull requests

Smaller is better. Open pull requests are a sign of of stalled work or slow reviews.

Cumulative count of opened issues

To be viewed in conjunction with the Count of open issues. We're more interested in the slope of each line, rather than the absolute numbers.

Cumulative count of opened pull requests

To be viewed in conjunction with the Count of open pull requests. We're more interested in the slope of each line, rather than the absolute numbers.

Cumulative count of unique users opening issues

The number of open GitHub issues in each repository.

This doesn't account for the number of issues opened by each user, just the date of their first issue.

Cumulative count of unique users opening pull requests

This doesn't account for the number of contributions by each user, just the date of their first contribution.
